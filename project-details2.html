Overview

The Online Quiz Application is a full-stack web platform developed using the MERN stack (MongoDB, Express.js, React.js, Node.js) that enables administrators to create and manage quizzes while allowing students to attend quizzes and view results instantly. The system is designed to support real-time evaluation, result analysis, and performance visualization through interactive charts and graphs.

Responsive design for all devices

Key Features

  • Provide a digital platform for conducting online quizzes
  • Enable secure admin and student authentication
  • Support real-time quiz evaluation
  • Analyze student performance using graphical reports
  • Improve learning efficiency through instant feedback
  • Secure Admin Login & Authentication
  • View student submissions
